# Hospedaje Sierra - Frontend

Sistema de gestión hotelera desarrollado en React JS + Vite.

## Tecnologías utilizadas
- React 18
- React Router v6
- Bootstrap 5
- Axios
- Context API para autenticación

## Instalación y ejecución
1. Clonar el repositorio:
git clone https://github.com/anamariasi1611-hash/hospedaje-sierra-frontend.git
2. Entrar a la carpeta:
cd hospedaje-sierra-frontend
3. Instalar dependencias:
npm install
4. Crear archivo `.env` en la raíz con:
VITE_API_URL=http://localhost:8080
5. Ejecutar en modo desarrollo:
npm run dev
6. Abrir http://localhost:5173 (o el puerto que muestre)
